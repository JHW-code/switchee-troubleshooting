Is the thermostat calling for heat? Is the boiler firing and radiators getting warm? Is the BRU receiving the heating command? Use the red override button on the BRU Boiler needs repair engineer visit Connectivity issue between thermostat and BRU Override button not working / urgent support needed

1. Is the thermostat calling for heat?

Ask the resident: Press the '+' button to increase the set temperature above the current room temperature. If the thermostat reacts, proceed to check if the boiler is firing.

2. Is the boiler firing and radiators getting warm?

Confirm if the boiler starts and radiators warm up. If yes, no further action needed. If no, continue troubleshooting.

3. Is the BRU receiving the heating command?

Ask the resident to check if the red light above the radiator symbol on the BRU is ON.

4. Use the red override button on the BRU

Ask the resident to press the red override button on the BRU to manually turn ON heating (look for flashing red/orange light).

5. Boiler needs repair engineer visit

If Switchee system is communicating but boiler is not responding, arrange a boiler repair engineer visit.

6. Connectivity issue between thermostat and BRU

If no light on BRU, use override button for temporary heating and organize a Switchee check/repair visit.

7. Override button not working / urgent support needed

Arrange urgent repair visit and advise resident to call Switchee at 0800 133 7957.
